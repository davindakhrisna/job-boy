## Cara memakai project ini

Yang diperlukan sebelum memulai hanya insstallasi Node.js dan npm

```sh
# Step 1
git clone <REPO_URL>

# Step 2
cd <REPO_URL>

# Step 3
npm i

# Step 4
npm run dev
```

## Technologies used in this project

Projek ini memakai:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
